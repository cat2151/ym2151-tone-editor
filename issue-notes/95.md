# issue キーリピートで1秒間に10回以上演奏したときに、演奏が反映されなくなる #95
[issues #95](https://github.com/cat2151/ym2151-tone-editor/issues/95)

# 原因
- clear scheduleしていたため
- 仮説
  - 未来オフセットが30msあるので、
  - clear scheduleすると、
  - 「さっき送信した演奏データが、30ms後にrender予定だったものが、renderされる前に、消えてしまう」

# 対策
- clear scheduleをコメントアウトして様子見

# 結果
- 問題なく、キーリピートで1秒間に10回以上、key onできて音が鳴るようになった

# closeとする
