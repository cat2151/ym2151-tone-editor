# Project Overview

## プロジェクト概要
- Rust製のTUI（テキストユーザーインターフェース）で、YM2151 FM音源のトーンパラメータを編集するためのエディタです。
- DetuneやMultiplierといった10種類のオペレーターパラメータを視覚的に表示し、Vimライクなキー操作で直感的に調整できます。
- FM音源の音色設計を支援し、ym2151-log-play-serverと互換性のある基本トーンを作成可能です。

## 技術スタック
- フロントエンド: `ratatui` (ターミナルUIフレームワーク)、`crossterm` (クロスプラットフォームなターミナル操作ライブラリ)
