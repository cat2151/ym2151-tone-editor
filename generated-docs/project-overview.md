# Project Overview

## プロジェクト概要
- YM2151 FM音源の音色パラメータを編集するためのRust製TUI (Text User Interface) エディタです。
- カーソル操作やキー入力によりパラメータを調整し、リアルタイムに音色を試奏できます。
- Windows環境での利用を想定しており、音色データをJSON形式で保存・読み込みが可能です。

## 技術スタック
- フロントエンド: `ratatui` (ターミナルUIフレームワークで、テキストベースのユーザーインターフェースを構築), `crossterm` (クロスプラットフォームなターミナル操作ライブラリで、カーソル制御、キーイベント処理などを提供)

## ファイル詳細説明
- `.gitignore`: Gitが追跡しないファイルやディレクトリを指定します。ビルド生成物や一時ファイルなどが含まれます。
- `Cargo.lock`: プロジェクトの依存関係の正確なバージョンを記録し、ビルドの再現性を保証します。
- `Cargo.toml`: Rustプロジェクトのマニフェストファイル。プロジェクト名、バージョン、著者情報、依存クレート、ターゲット設定などが定義されています。
- `LICENSE`: プロジェクトの利用条件を定めるライセンス情報が記述されています。
- `NOTE_ON_VISUALIZATION.md`: プロジェクトにおける視覚化に関する考察やメモを記録したMarkdownファイルです。

- `_config.yml`: 静的サイトジェネレータ（例: Jekyll）でドキュメントサイトを構築する際の設定ファイルである可能性があります。
- `generated-docs/`: プロジェクトから生成されたドキュメントやレポートなどが格納されるディレクトリです。
- `issue-notes/`: 開発中に発見された課題、設計上の検討事項、ノウハウなどを記録した個別のメモファイル群が格納されています。
- `src/main.rs`: Rustアプリケーションのエントリポイントとなる主要なソースコードファイルです。TUIの描画ロジック、YM2151パラメータの編集処理、キーボード入力などのイベント処理が実装されています。

## 関数詳細説明
このプロジェクトでは、主に以下の役割を持つ関数群が実装されていると推測されます。

- **`main`関数**: アプリケーションの開始点。ターミナルの初期設定、TUIアプリケーションのメインループの実行、および終了時のターミナル状態の復元を担当します。
- **UI描画関連関数**: 現在のYM2151音色パラメータの状態やユーザーインターフェース要素（パラメータ名、値、カーソル位置など）をターミナル画面上に描画するロジックをカプセル化しています。
- **イベント処理関数**: キーボード入力やマウスイベント（提供情報にマウス操作あり）を監視し、それに応じてアプリケーションの状態（カーソルの移動、パラメータ値の増減、アプリケーションの終了など）を更新します。
- **YM2151パラメータ管理関数**: YM2151の11種類の音色パラメータ（DT, MUL, TLなど）と5つのオペレーター/チャンネル（OP1-4, CH）の値を内部的に保持・管理します。値の検証（範囲制限）や、将来的にJSON形式への変換、外部システムへの送信処理なども含まれます。
- **ターミナル制御関数**: `crossterm`ライブラリを利用し、ターミナルをrawモードに設定したり、カーソルを非表示にしたり、画面をクリアしたりといった低レベルなターミナル操作を行います。
