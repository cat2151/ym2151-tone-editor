# Project Overview

## プロジェクト概要
- YM2151（OPM）FM音源の音色を編集するための、Windows向けターミナルユーザーインターフェース（TUI）エディタです。
- 直感的な操作性（マウス・キーボード）で音色パラメータの調整ができ、リアルタイムでサウンドをプレビューできます。
- シンプルな設計思想のもと、素早い起動と応答性、そして視覚的に分かりやすいインターフェースを提供します。

## 技術スタック
- フロントエンド: `ratatui` (ターミナルUIフレームワーク) - ターミナル上でリッチなユーザーインターフェースを構築するためのフレームワークです。
- フロントエンド: `crossterm` (クロスプラットフォームターミナル操作) - ターミナル上でのキー入力、マウスイベント、カーソル移動など、低レベルな操作をクロスプラットフォームで実現するライブラリです。
- 音楽・オーディオ: `ym2151-log-play-server` (リアルタイム音声フィードバック) - YM2151の音色データをリアルタイムでオーディオ再生するためのサーバー連携ライブラリです。
- 音楽・オーディオ: `mmlabc-to-smf-rust` (MMLからSMF変換) - プレビューMML（Music Macro Language）をStandard MIDI File (SMF) 形式に変換するライブラリです。
- 音楽・オーディオ: `smf-to-ym2151log-rust` (SMFからYM2151ログ変換) - SMFと音色データからYM2151のレジスタログ形式に変換するライブラリです。
- 開発ツール: `Rust` (プログラミング言語) - パフォーマンス、安全性、並行性に優れたシステムプログラミング言語で、本プロジェクトの主要な開発言語です。
- 開発ツール: `cargo` (Rustのビルドシステム・パッケージマネージャー) - Rustプロジェクトのビルド、依存関係管理、テスト実行などを行う標準ツールです。
- テスト: Rustのテストフレームワーク - Rustに組み込まれたテスト機能を利用し、コードの品質と信頼性を確保します。
- ビルドツール: `cargo` - プロジェクトのコンパイルと実行を管理します。
- 言語機能: Rustの所有権システム、ライフタイム、安全性 - メモリ安全性を保証し、バグの少ない堅牢なアプリケーション開発に貢献するRustの主要な機能群です。
