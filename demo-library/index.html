ym2151-wasm demo

YM2151 (OPM) FM音源のランダムトーンをWASMで生成するデモです。
generate_random_tone_registers(seed, note) を呼び出し、 レジスタデータ（16進数文字列）を取得します。

ランダムトーン生成

（ボタンを押してください）

利用例 (TypeScript / JavaScript)

// pkg/ を同じディレクトリに配置してから import する
import init, { generate_random_tone_registers } from './pkg/ym2151_wasm.js';

await init();

const seed = Date.now();          // シード値
const note = 69;                  // MIDI ノート番号 (69 = A4)
const registers = generate_random_tone_registers(seed, note);
// 例: "2000280060001F607F..."  (4文字ごとに addr+data)
console.log(registers);

レジスタ形式

戻り値は 4文字単位 の16進数文字列です。
先頭2文字がレジスタアドレス、後ろ2文字がデータです。 

例: "20002800600080009F00..."
    ^^^^    → addr=0x20, data=0x00  (ALG/FB/PAN)
        ^^^^  → addr=0x28, data=0x00  (KC)
            ^^^^→ addr=0x60, data=0x00  (TL OP1)
